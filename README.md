# Modus Tollens
2D Realtime economic strategy game with no main objectives.

## Prerequisites
* Linux or Mac OS X
* SFML 2.3.2
* boost 1.60.0
* gtest 1.7.0
* cmake 3.5.0
* gcc

## Compilation with cmake

    ./mt-install

or

    mkdir Build
    cd Build
    cmake -G "Unix Makefiles"
    make

## Run game

    ./modus_tollens
