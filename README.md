# Modus Tollens
Gra strategiczna wzorowana na Dwarf Fortress.

## Wymagania

* SFML 2.3.2
* boost 1.60.0
* gtest 1.7.0
* cmake 3.5.0
* gcc

## Kompilacja z użyciem cmake

    mkdir Build
    cd Build
    cmake -G "Unix Makefiles"
    make

## Uruchomienie testów (gtest)

    ./modus_tollens_test

## Uruchomienie gry

    ./modus_tollens
